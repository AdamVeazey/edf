Heap<T, N, Compare>

Template arguments

T = (T)ype
N = Maximum (N)umber of elements the heap can hold
Compare = (Compare) functor used to specify how elements should be sorted

T Needs to be copyable, and helps if T is also default constructable.

Overview

This is a fixed max size binary heap container. Providing different Compare functors allows you to choose how the heap is sorted. There are two pre-defined aliases HeapMax<T, N> and HeapMin<T, N>.

HeapMax<T, N>

This is an alias for a binary max heap.

template<typename T, std::size_t N>
using HeapMax = Heap<T, N, std::greater<T>>;

HeapMin<T, N>

This is an alias for a binary min heap.

template<typename T, std::size_t N>
using HeapMin = Heap<T, N, std::less<T>>;

Initialization

Default initialization requires type T to be default constructable. Otherwise list-initialization is required.

Example: T is default constructable
EDF::HeapMax<int, 7> heap;
Example: list-initialization with a custom type
EDF::HeapMax<int, 7> heapInitializationList = { 10, 20 };
If using list-initialization on a type T without a default constructor. First fill up the entire heap using list-initialization, then call clear()

Is Questions

These member functions provide yes/no answers to the current state of the heap.

isEmpty()

Equivalent to length() == 0
Provided to make a common conditional check slightly more readable, as well as consistency with other EDF containers.

Example
EDF::HeapMax<int, 7> heap;
while( !heap.isEmpty() ) {
    /* ... */

isFull()

Since a heap doesn’t grow the underlying container, you should check if it is full before adding an element.

Example
EDF::HeapMax<int, 7> heap;
if( !heap.isFull() ) {
    /* ... */
}

Capacity

Provides information about how many current elements are in the heap, as well as the max number of elements the heap can hold.

length()

Returns the number of elements currently in the heap.

Example
EDF::HeapMax<int, 7> heap;
auto& nElementsInHeap = heap.length();

maxLength()

Returns template parameter N - 1. It acts the same as a compile time constant. No need to remember how a macro for the heap max number of elements is named.

Example
EDF::HeapMax<int, 7> heap;
constexpr auto maxNumberOfElements = heap.maxLength();

Operations

Many, if not all, of the Operation member functions use EDF_ASSERTD to ensure the operation will be successful. Remember that the assert condition is only checked in Debug mode.

peek()

Returns the value at the root of the heap without modifying the number of elements in the heap.

Example
EDF::HeapMax<int, 7> heap;
heap.push( 2 );
auto& rootElement = heap.peek();

push( value )

Add element to the the heap.

Example
EDF::HeapMax<int, 7> heap;
heap.push( 2 );

emplace( args…​ )

Add element to the heap by constructing the element in place using placement new.

Example
EDF::HeapMax<int, 7> heap;
heap.emplace( 6 );

pop()

Remove element from the heap then return a copy of the value.

Example
EDF::HeapMax<int, 7> heap;
    auto removedElement = heap.pop();

clear()

Calls destructor for each element in heap. Sets the number of elements to 0.

Example
EDF::HeapMax<int, 7> heap;
heap.clear();
