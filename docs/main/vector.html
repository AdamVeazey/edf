Vector<T, N>

Template arguments

T = (T)ype
N = Maximum (N)umber of elements the vector can hold

T Needs to be copyable, and helps if T is also default constructable.

Overview

Vector is a template that uses an Array plus a current size variable to act like a std::vector, but does not grow if there is no more room in the underlying Array. N specifies the maximum number of elements the vector can "grow" to.

There are 5 groups of member functions:

  1. Is Questions

  2. Capacity

  3. Element Access

  4. Operations

  5. Iterators

When using overloaded member functions that use either index/position, when using an index of 0 will attempt to use the position version of the member function. To work around this, use _uz for 0. All other integers will have expected overload resolution.

Initialization

Default initialization requires type T to be default constructable. Otherwise list-initialization is required.

Example: Default Constructable
EDF::Vector<int, 30> vector;
Example: list-initialization with a custom type
class CustomType {
private:
    int x;
    int y;
public:
    CustomType( int x, int y = 0 ) : x(x), y(y) {}
    // ...
};

EDF::Vector<CustomType, 3> vector = {
    CustomType(1, 1),
    CustomType(2, 2),
    CustomType(3, 3)
};

Is Questions

These member functions provide yes/no answers to the current state of the vector.

isEmpty()

Equivalent to length() == 0
Provided to make a common conditional check slightly more readable, as well as consistency with other EDF containers.

Example
if( myVector.isEmpty() ) { /* ... */}
// vs.
if( myVector.length() == 0 ) { /* ... */ }

isFull()

Since a vector doesn’t grow, you should check if it is full before adding an element. If the vector is full this function return true. Otherwise it returns false.

Example
if( !myVector.isFull() ){
    myVector.pushBack( value );
}

Capacity

Provides information about how many current elements are in the vector, as well as the max number of elements in the vector.

length()

Returns the number of elements currently in the vector.

Example
EDF::Vector<int, 16> vector = { 10, 2, 3, 4, 5, 6 };    
std::size_t numberOfElementsInVector = vector.length(); (1)
1 The value of numberOfElementsInVector is 6. As opposed to maxLength() which returns 16.
The name length() is used instead of size() or count() for consistency across all EDF containers and String

maxLength()

Returns template parameter N. It acts the same as a compile time constant. No need to remember how a macro for a vector max number of elements is named.

Example
EDF::Vector<int, 16> vector = { 10, 2, 3, 4, 5, 6 };    
static constexpr int someArray[vector.maxSize()]; (1)
1 someArray will be able to hold 16 elements, since vector.maxSize() is 16

Element Access

operator[]

This is exactly the same as with a C-style array.

Example
EDF::Vector<int, 16> vector = { 10, 2, 3, 4, 5, 6 };    
vector[0_uz] = 100;
This also means that there is no boundary checking, so it is possible to attempt to access data outside of the vector.
0_uz

at( index )

index is checked with EDF_ASSERTD to be sure index is valid.

Example
EDF::Vector<int, 16> vector = { 10, 2, 3, 4, 5, 6 };    
vector.at( 0_uz ) = 1;
0_uz

front()

Return the first element of the vector. Calls at( index )

Example
EDF::Vector<int, 16> vector = { 10, 2, 3, 4, 5, 6 };    
auto& firstElement = vector.front();

back()

Return the last element of the vector (length() - 1). Calls at( index )

Example
EDF::Vector<int, 16> vector = { 10, 2, 3, 4, 5, 6 };    
auto& lastElement = vector.back();

data()

Returns a raw pointer to the first element of the vector.

Example
void working_with_a_c_library_function( int *data, size_t n ) { /* ... */}

EDF::Vector<int, 16> vector = { 10, 2, 3, 4, 5, 6 };    
working_with_a_c_library_function( vector.data(), vector.length() );
Can be called in order to "decay" to a pointer like a C-Style array

Operations

Many, if not all, of the Operation member functions use EDF_ASSERTD to ensure the operation will be successful. Remember that the assert condition is only checked in Debug mode.
For member functions that have an index/position as their first argument, it means that overloads exist to use an index (std::size_t) or an iterator.

clear()

Calls destructor for each element in vector. Sets number of elements to 0.

Example
EDF::Vector<int, 16> vector = { 10, 2, 3, 4, 5, 6 };    
vector.clear();
std::size_t numberOfElementsInVector = vector.length(); (1)
1 numberOfElementsInVector value is 0. ~int() called for each element that was in the vector.

insert( index/position, value )

Inserts value at the specified index or position.

Example
EDF::Vector<int, 16> vector = { 10, 2, 3, 4, 5, 6 };    
vector.insert( 0_uz, 0 );           (1)
vector.insert( vector.begin(), 0 ); (2)
1 Index usage NOTE: 0_uz
2 Iterator usage

insert( index/position, count, value )

Inserts value for count number of times starting at the specified index or position.

Example
EDF::Vector<int, 16> vector = { 10, 2, 3, 4, 5, 6 };    
vector.insert( 0_uz, 2, 0 );            (1)
vector.insert( vector.begin(), 2, 0 );  (2)
1 Index usage to insert 2 `0’s at the beginning of the vector. NOTE: 0_uz
2 Iterator usage. In this particular example, EDF_ASSERTD will catch the fact that the vector is already full.

insert( index/position, { values…​ } )

Inserts the list of values…​ provided starting at the specified index or position.

Example 1
EDF::Vector<int, 16> vector = { 10, 2, 3, 4, 5, 6 };    
vector.insert( 0_uz, { 0 } );           (1)
vector.insert( vector.begin(), { 0 } ); (2)
1 Index usage NOTE: 0_uz
2 Iterator usage
Example 2
EDF::Vector<int, 16> vector = { 10, 2, 3, 4, 5, 6 };    
for( auto it = vector.cbegin(); it != vector.cend(); ++it ) {
    if( *it == 3 ) {
        vector.insert( it, {99, 88, 77, 66, 55} );
        it += 5;
    }
}

emplace( index/position, args…​ )

Overwrites an element by using placement new and args…​ to construct the an element in place.

Example
class CustomType {
private:
    int x;
    int y;
public:
    CustomType( int x = 0, int y = 0 ) : x(x), y(y) {}
    // ...
};

EDF::Vector<CustomType, 3> vector2;
vector2.emplace( 0_uz, 1, 1 );  (1)
vector2.emplace( 1, 2, 2 );     (2)
1 Constructs a CustomType at the first index of the vector. NOTE: 0_uz
2 Constructs a CustomType at the second index of the vector

erase( index/position )

Erase a single element at the specified index or position. Calls destructor for erased element.

Example
EDF::Vector<int, 16> vector = { 10, 2, 3, 4, 5, 6 };    
vector.erase( vector.length() - 2 ); 
vector.erase( std::find( vector.cbegin(), vector.cend(), 12345 ) );

erase( first, last )

Erase a range [first, last) of elements. Includes erasing first up to, but not including, last. Calls destructor for erased elements.

Example
EDF::Vector<int, 16> vector = { 10, 2, 3, 4, 5, 6 };    
vector.erase( 2, 4 );    // [2,4) 
auto seventySeven = std::find(vector.cbegin(), vector.cend(), 77);
vector.erase( seventySeven, seventySeven + 3 );

pushBack( value )

Copy value to a new element at the end of the vector.

Example
EDF::Vector<int, 16> vector = { 10, 2, 3, 4, 5, 6 };    
vector.pushBack( 2 + 2 );
vector.pushBack( 12345 );

emplaceBack( args…​ )

Construct in place a new element at the end of the vector using args…​.

Example
EDF::Vector<int, 16> vector = { 10, 2, 3, 4, 5, 6 };    
vector.emplaceBack( 100 );

popBack()

Remove the last element in a vector and return a copy of it.

Example
EDF::Vector<int, 16> vector = { 10, 2, 3, 4, 5, 6 };    
auto copyOfLastElement = vector.popBack();

operator==( lhs, rhs )

lhs = left hand side
rhs = right hand side
Compare two vectors to see if each element in lhs and rhs are equal.

Iterators

Forward iterators and const iterators are available through begin(), end(), cbegin(), cend()

Revese iterators and const reverse iterators are available through rbegin(), rend(), crbegin(), crend()

Range-based for loops are therefore supported.
