Color

Overview

Color is a 32-bit value where 8-bits are dedicated to red, green, blue, and alpha. Color565 is a 16-bit value where 5 bits are dedicated to red, 6 for blue, and 5 for blue. Pre-defined named colors are static members of Color.

Initialization

Color(), Color565()

EDF::Color black;
EDF::Color565 black;

Color( initialValue ), Color565( initialValue )

EDF::Color salmon( 0xFA8072 );
EDF::Color565 salmon( 0xFC0E );

Color( r, g, b, a )

a is optional, default’s to 255 if not provided which means opacity is 100%.

EDF::Color white( 255, 255, 255 );
EDF::Color gray( 128, 128, 128, 255 );

Color565( r, g, b )

EDF::Color565 white( 255, 255, 255 );

Color565( c )

c is a Color

EDF::Color565 color( EDF::Color::blueViolet() );

Set Color Components

setR( r ), setG( g ), setB( b ), setA( a )

setA( a ) only exists for Color not Color565. 
EDF::Color black;
black.setR( 255 );  // turn red
black.setG( 255 );  // turn yellow
black.setB( 255 );  // turn white
black.setA( 0 );    // turn transparent

EDF::Color565 black;
black.setR( 255 );  // turn red
black.setG( 255 );  // turn yellow
black.setB( 255 );  // turn white

Get Color Components

r(), g(), b(), a()

a() only exists for Color not Color565. 
EDF::Color salmon( 0xFA8072 );
salmon.r(); // 0xFA
salmon.g(); // 0x80
salmon.b(); // 0x72
salmon.a(); // 0xFF

EDF::Color565 salmon( 0xFC0E );
salmon.r(); // 0xF8
salmon.g(); // 0x80
salmon.b(); // 0x70

asRGB(), asRGBA()

asRGBA() only exists for Color not Color565. 
EDF::Color white( 255, 255, 255 );
EDF::Color gray( 128, 128, 128, 255 );
white.asRGB();  // 0xFFFFFF
gray.asRGBA();  // 0xFFFFFFFF

EDF::Color565 white( 255, 255, 255 );
white.asRGB();  // 0xFFFF

Named Colors

There are a few named colors predefined in Color.

Predefined Colors
static constexpr Color red()        { return Color( 255,   0,   0 ); }
static constexpr Color orange()     { return Color( 255, 128,   0 ); }
static constexpr Color yellow()     { return Color( 255, 255,   0 ); }
static constexpr Color lime()       { return Color( 128, 255,   0 ); }
static constexpr Color green()      { return Color(   0, 255,   0 ); }
static constexpr Color springGreen(){ return Color(   0, 255, 128 ); }
static constexpr Color cyan()       { return Color(   0, 255, 255 ); }
static constexpr Color dodgerBlue() { return Color(   0, 128, 255 ); }
static constexpr Color blue()       { return Color(   0,   0, 255 ); }
static constexpr Color blueViolet() { return Color( 128,   0, 255 ); }
static constexpr Color magenta()    { return Color( 255,   0, 255 ); }
static constexpr Color deepPink()   { return Color( 255,   0, 128 ); }

static constexpr Color black()	    { return Color(   0,   0,   0 ); }
static constexpr Color dimGray()	{ return Color( 105, 105, 105 ); }
static constexpr Color gray()	    { return Color( 128, 128, 128 ); }
static constexpr Color darkGray()	{ return Color( 169, 169, 169 ); }
static constexpr Color silver()	    { return Color( 192, 192, 192 ); }
static constexpr Color lightGray()	{ return Color( 211, 211, 211 ); }
static constexpr Color gainsboro()	{ return Color( 220, 220, 220 ); }
static constexpr Color whiteSmoke()	{ return Color( 245, 245, 245 ); }
static constexpr Color white()	    { return Color( 255, 255, 255 ); }
Example Syntax to Access Predefined Colors
EDF::Color::red();
EDF::Color::springGreen();
EDF::Color::cyan();
Example of how to use Color565 and Predefined Colors
void function( const EDF::Color565& color ) {
    /* ... */
}
 function( EDF::Color::magenta() );
