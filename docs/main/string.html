String<N>

Template arguments

N = Maximum (N)umber of characters the string can hold.

N includes '\0', so if N = 8 the string can hold 7 characters, with the remaining spot used to store a null terminator ('\0')

TL;DR

The following example shows some usage of string operations to check if string is a palindrome or not without mutating the original string.

EDF::String<32> string = "Madam in Eden, Im Adam.";
auto normalized = string.getToLower().strip( " ,." );
bool isPalindrome = normalized.equals( normalized.getReversed() );

Overview

String is a a fixed max size character array that has many operations. At its core a string is a std::size_t length + a char buffer[N], where buffer[length] always has the value '\0'. This class is provided to make common string manipulations using a character array.

There are three big picture concepts that will help make sense of this large class.

  1. Many operations have a mutating and non-mutating version. The naming convention is to have the mutating version be named "presentTenseOperation" and the non-mutating version be named "getPastTenseOperation". For example, append( value ) and getAppended( value ).

  2. Common list of overloads for many functions. The following types listed below are supported by many operations.

    1. f( const char* str );

    2. f( const uint8_t* str );

    3. f( const char* str, std::size_t n );

    4. f( const uint8_t* str, std::size_t n );

    5. f( char ch );

    6. f( int8_t value, int base = 10 );

    7. f( int16_t value, int base = 10 );

    8. f( int32_t value, int base = 10 );

    9. f( int64_t value, int base = 10 );

    10. f( uint8_t value, int base = 10 );

    11. f( uint16_t value, int base = 10 );

    12. f( uint32_t value, int base = 10 );

    13. f( uint64_t value, int base = 10 );

    14. f( const String<S>& o );

  3. Overload resolution. Except for std::size_t n which indicates parameter string length, every other std::size_t argument should be of the correct type. To aid with this you can use _uz.

There are 6 groups of member functions:

  1. Initialization

  2. Is Questions

  3. Capacity

  4. Element Access

  5. Operations

  6. Iterators

Is Questions

These member functions provide yes/no answers to the current state of the string.

isEmpty()

Equivalent to length() == 0
Provided to make a common conditional check slightly more readable, as well as consistency with other EDF containers.

Example
EDF::String<128> string = "Hello, world!";
if( !string.isEmpty() ) {
    /* ... */
}

isFull()

Since a string doesn’t grow the underlying container. Returns if length() == N - 1

Example
EDF::String<128> string = "Hello, world!";
if( !string.isFull() ) {
    /* ... */
}

Capacity

Provides information about how many current characters are in the string, as well as the max number of characters the string can hold.

length()

Returns the number of characters currently in the string.

Example
EDF::String<128> string = "Hello, world!";
auto nCharactersInString = string.length();

maxLength()

Returns template parameter N - 1. It acts the same as a compile time constant. No need to remember how a macro for the string max number of characters is named.

Example
EDF::String<128> string = "Hello, world!";
constexpr auto maxNumberOfCharacters = string.maxLength();

Element Access

Operations

Many operations have a mutating and non-mutating version. The naming convention is to have the mutating version be named "presentTenseOperation" and the non-mutating version be named "getPastTenseOperation". For example, append( value ) and getAppended( value ).

Many, if not all, of the operation member functions use EDF_ASSERTD to ensure the operation will be successful. Remember that the assert condition is only checked in Debug mode.
For member functions that have an index/position as their first argument, it means that overloads exist to use an index (std::size_t) or an iterator.
High level list of operations available

  1. append( value )

  2. insert( index/position, value )

  3. erase( index/position )

  4. clear()

  5. copyTo( outputString, maxBufferLength )

  6. find( value )

  7. rfind( value )

  8. contains( value )

  9. equals( value )

  10. strip( value )

  11. trim( value )

  12. trimLeft( value )

  13. trimRight( value )

  14. reverse()

  15. toLower()

  16. toUpper()

  17. replace( lookFor, replaceWith )

  18. subString( start, end )

  19. +=( rhs )

  20. +( this, rhs )

append( value )

Appends the value to the end of the string, given there is enough space to hold that value. All of the overloads for different values exist for append( value ). This set of member functions do mutate string.

Example
EDF::String<128> string = "Hello, world!";
string.append( "Hello, world!" );
string.append( 1234 );
string.append( "0x" ).append( 0x5678, 16 );

getAppended( value )

Makes a copy of the current string, then appends value to the copied string. All of the overloads for different values exist for getAppended( value ). This set of member functions do NOT mutate string.

Example
EDF::String<128> string = "Hello, world!";
string.getAppended( "Hello, world!" );
string.getAppended( 1234 );
string.getAppended( "0x" ).append( 0x5678, 16 );

insert( index/position, value )

Inserts the value to at the specified index/position, given there is enough space to hold that value. All of the overloads for different values exist for insert( index/position, value ), with the additional index/position parameter as the first argument. This set of member functions do mutate string.

Example
EDF::String<128> string = "Hello, world!";
string.insert( 0_uz, "Hello, world!");
string.insert( 6_uz, 4 );
string.insert( 7, 0x2, 16 );
string.insert( string.end() - 1, "s and solar system" );
string.insert( 6, ' ' );

getInserted( index/position, value )

Makes a copy of the current string, then inserts value to the copied string at the specified index/position. All of the overloads for different values exist for getInserted( index/position, value ), with the additional index/position parameter as the first argument. This set of member functions do NOT mutate string.

Example
EDF::String<128> string = "Hello, world!";
string.getInserted( 6_uz, 4 );
string.getInserted( 7, 0x2, 16 );
string.getInserted( string.end() - 1, "s and solar system" );
string.getInserted( 6, ' ' );

erase( index/position )

Erase a single character at the selected index/position. This member function does mutate the string.

Example
EDF::String<128> string = "Hello, world!";
string.erase( 5 );

getErased( index/position )

Makes a copy of the current string, then erases a single character at the selected index/position. This member function does NOT mutate the string.

Example
EDF::String<128> string = "Hello, world!";
string.getErased( 5 );

erase( start index/position, end index/position )

Erase a range of characters [start, end) starting from start up to but not including end. This member function does mutate the string.

Example
EDF::String<128> string = "Hello, world!";
string.erase( 5, 12 );

getErased( start index/position, end index/position )

Makes a copy of the current string, then erases a range of characters [start, end) starting from start up to but not including end. This member function does NOT mutate the string.

Example
EDF::String<128> string = "Hello, world!";
string.getErased( 5, 12 );

clear()

Erases everything in the string, such that length() returns 0. Underlying buffer still contains a null terminator.

strip( value )

If value is provided, all occurrences of each character provided are removed from the string. If no value is provided, strip removes all non-printable characters.

Non-printable characters are the values in the following ranges [0x00, 0x1F] and [0x7F, 0xFF]. Printable characters are defined as a character that meets the following condition: `character > ' ' && character ⇐ '~'`.
Example
EDF::String<128> string = "Hello, world!";
string.strip();
string.strip( '\n' );
string.strip( " ,\t\n" );
string.strip( EDF::String<4>("lo") );

strip( value, n )

If value is provided, all occurrences of each character provided are removed from the string. If no value is provided, strip removes all non-printable characters.

Non-printable characters are the values in the following ranges [0x00, 0x1F] and [0x7F, 0xFF]. Printable characters are defined as a character that meets the following condition: `character > ' ' && character ⇐ '~'`.
Example
EDF::String<128> string = "Hello, world!";
string.strip( " \x1B", std::strlen("\x1B") );

trim( value )

If no value is provided, trim removes all non-printable characters from this string [begin(), first printable character] as well as [last printable character, end()]. If a value is provided, occurrences of each character provided are removed from the same two ranges in the string.

Non-printable characters are the values in the following ranges [0x00, 0x1F] and [0x7F, 0xFF]. Printable characters are defined as a character that meets the following condition: `character > ' ' && character ⇐ '~'`.
Example
EDF::String<128> string = "Hello, world!";
string = "\t\x1B Hello, world! \r\n ";
string.trim( '\t' ); // "\x1B Hello, world! \r\n "
string.trim( " \r\n" ); // "\t\x1B Hello, world!"
string.trim(); // "Hello, world!"

string = "\t\x1B Hello, world! \r\n ";
string.trim( "\r \n\x1B", std::strlen("\r \n\x1B") ); // "\t\x1B Hello, world!"

getTrimmedLeft( value )

If no value is provided, trim removes all non-printable characters from a copy of this string [begin(), first printable character]. If a value is provided, occurrences of each character provided are removed from the same range in the string.

Non-printable characters are the values in the following ranges [0x00, 0x1F] and [0x7F, 0xFF]. Printable characters are defined as a character that meets the following condition: `character > ' ' && character ⇐ '~'`.
Example

string = "\t\x1B Hello, world! \r\n ";
string.getTrimmedLeft( '\t' ); // "\x1B Hello, world! \r\n "
string.getTrimmedLeft( "\r\n" ); // "\t\x1B Hello, world! \r\n "
string.getTrimmedLeft(); // "Hello, world! \r\n "

Range-based for loops are therefore supported.
