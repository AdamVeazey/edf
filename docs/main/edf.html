EDF

Overview

The files here are written to be completely platform independent that are useful in any embedded systems project. Within this collection of files are two main groups

  1. STL-Like Containers - Containers with a fixed max size that are statically allocated

  2. Miscellaneous - Classes that don’t fit into another group

STL-Like Containers

  1. Array - wrapper around regular array. Almost exactly the same as std::array

  2. Vector - Array that can "grow" up to a maximum size

  3. Stack - adapts EDF::Vector to turn it into an EDF::Stack

  4. Queue - circular queue (AKA ring buffer) using an EDF::Array

  5. Heap - min and max heap using an EDF::Vector

Miscellaneous

  1. Assert - assert a condition is true, abort() if false

  2. String - Strings that can "grow" up to a maximum size

  3. Version - String to semantic version for easy comparisons

  4. Math - A collection of MISC common math functions

    • _uz

    • nElements<T,N>

    • isPow2

    • min<T>

    • max<T>

    • voltageDividerCalcVout<R1, R2, T>

    • voltageDividerCalcVin<R1, R2, T>

    • temperatureCtoF<T>

    • temperatureFtoC<T>

  5. BitField - Name one or more specific bits within an unsigned integer

  6. Color - RGBA colors

  7. UnixTimestamp - wrapper for <ctime>
