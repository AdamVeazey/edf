Queue<T, N>

Template arguments

T = (T)ype
N = Maximum (N)umber of elements the underlying container can hold. The queue can hold at most (N)-1 number of elements.

The queue can hold at most N-1 number of elements.
T Needs to be copyable, and helps if T is also default constructable.

Overview

This is a container to provide a FIFO data structure with a fixed max size. Since queue has a fixed max size, it is implemented as a circular queue, aka ring buffer.

There are 4 main groups of member functions:

  1. Is Questions

  2. Capacity

  3. Operations

  4. uint8_t Specialized Member Functions

Initialization

Default initialization requires type T to be default constructable. Otherwise list-initialization is required.

Example: T is default constructable
EDF::Queue<int, 8> queue;
Example: list-initialization with a custom type
struct CustomType{
    int value;
    CustomType( int v ) : value(v) {}
};

EDF::Queue<CustomType, 4> queue2 = {
    CustomType( 0 ),
    CustomType( 0 ),
    CustomType( 0 ),
    CustomType( 0 )
};
queue2.clear();
If using list-initialization on a type T without a default constructor. First fill up the entire queue using list-initialization, then call clear()

Is Questions

These member functions provide yes/no answers to the current state of the queue.

isEmpty()

Equivalent to length() == 0
Provided to make a common conditional check slightly more readable, as well as consistency with other EDF containers.

Example
EDF::Queue<int, 8> queue;
while( !queue.isEmpty() ){
    /* ... */

isFull()

Since a queue doesn’t grow the underlying container, you should check if it is full before adding an element.

Example
EDF::Queue<int, 8> queue;
if( !queue.isFull() ) {
    /* ... */
}

Capacity

Provides information about how many current elements are in the queue, as well as the max number of elements the queue can hold.

length()

Returns the number of elements currently in the queue.

Example
EDF::Queue<int, 8> queue;
auto nElementsInQueue = queue.length();

maxLength()

Returns template parameter N - 1. It acts the same as a compile time constant. No need to remember how a macro for the queue max number of elements is named.

Example
EDF::Queue<int, 8> queue;
constexpr auto maxNumberOfElements = queue.maxLength();

Operations

Many, if not all, of the Operation member functions use EDF_ASSERTD to ensure the operation will be successful. Remember that the assert condition is only checked in Debug mode.

peek()

Returns the first value inserted into the queue without modifying the number of elements in the queue.

Example
EDF::Queue<int, 8> queue;
queue.push( 1 );
auto& firstElementPeek = queue.peek();
Name is peek instead of "front" for consistency with Stack. It gives a better picture of what it does. Peek allows you to peek, look, read, the element you are likely to remove next using pop() without actually popping the value.

push( value )

Add element to the end of the queue.

Example
EDF::Queue<int, 8> queue;
queue.push( 1 );

emplace( args…​ )

Add element to the end of the queue by constructing the element in place using placement new.

Example
EDF::Queue<int, 8> queue;
auto& valueConstructed = queue.emplace( 2 );

pop()

Remove element from the end of the queue then return a copy of the value.

Example
EDF::Queue<int, 8> queue;
    auto firstElement = queue.pop();

clear()

Calls destructor for each element in queue. Sets the number of elements to 0.

Example
EDF::Queue<int, 8> queue;
queue.clear();

uint8_t Specialized Member Functions

A common usage of a queue is to hold a buffer of incoming data from a data source, like UART for example. Add incoming data to the queue with push( value ) and parse that data using pop(). The following set of functions are provided as alternatives to pop() when parsing an integer from the stream of data.

There are two groups of uint8_t Specialized Member Functions, big endian (AKA Network Order) and little endian. Big endian member functions are marked with the suffix "be". Little endian member functions are marked with the suffix "le".

Big Endian pop() Member Functions

pop8be()

Alias for pop(), only provided for consistency of other big endian pop functions.

Example
EDF::Queue<uint8_t, 32> q;
q.push( 0x12 );
uint8_t data8be = q.pop8be(); // 0x12

pop16be()

Calls pop8be() twice to get two bytes from the queue and interpret them as a big-endian 16-bit unsigned integer.

Example
EDF::Queue<uint8_t, 32> q;
q.push( 0x12 );
q.push( 0x34 );
uint16_t data16be = q.pop16be(); // 0x1234

pop32be()

Calls pop16be() twice to get four bytes from the queue and interpret them as a big-endian 32-bit unsigned integer.

Example
EDF::Queue<uint8_t, 32> q;
q.push( 0x12 );
q.push( 0x34 );
q.push( 0x56 );
q.push( 0x78 );
uint32_t data32be = q.pop32be(); // 0x12345678

pop64be()

Calls pop32be() twice to get eight bytes from the queue and interpret them as a big-endian 64-bit unsigned integer.

Example
EDF::Queue<uint8_t, 32> q;
q.push( 0x12 );
q.push( 0x34 );
q.push( 0x56 );
q.push( 0x78 );
q.push( 0x9A );
q.push( 0xBC );
q.push( 0xDE );
q.push( 0xF0 );
uint64_t data64be = q.pop64be(); // 0x123456789ABCDEF0

Little Endian pop() Member functions

pop8le()

Alias for pop(), only provided for consistency of other little endian pop functions.

Example
EDF::Queue<uint8_t, 32> q;
q.push( 0x12 );
uint8_t data8le = q.pop8le(); // 0x12

pop16le()

Calls pop8le() twice to get two bytes from the queue and interpret them as a little-endian 16-bit unsigned integer.

Example
EDF::Queue<uint8_t, 32> q;
q.push( 0x34 );
q.push( 0x12 );
uint16_t data16le = q.pop16le(); // 0x1234

pop32le()

Calls pop16le() twice to get four bytes from the queue and interpret them as a little-endian 32-bit unsigned integer.

Example
EDF::Queue<uint8_t, 32> q;
q.push( 0x78 );
q.push( 0x56 );
q.push( 0x34 );
q.push( 0x12 );
uint32_t data32le = q.pop32le(); // 0x12345678

pop64le()

Calls pop32le() twice to get eight bytes from the queue and interpret them as a little-endian 64-bit unsigned integer.

Example
EDF::Queue<uint8_t, 32> q;
q.push( 0xF0 );
q.push( 0xDE );
q.push( 0xBC );
q.push( 0x9A );
q.push( 0x78 );
q.push( 0x56 );
q.push( 0x34 );
q.push( 0x12 );
uint64_t data64le = q.pop64le(); // 0x123456789ABCDEF0
