Version

Overview

Version is a class to handle semantic version numbers, AKA "SemVer". The current version of EDF being used can be found with EDF Version

There are a few important limitations to be aware of.

  1. Version is able to support up to 3 digits per element.

  2. Version does not support pre-release

  3. Version does not support build metadata.

There are 4 groups of member functions:

  1. Initialization

  2. Access

  3. EDF Version

  4. Comparison Operators

Initialization

Version()

Default version number of 0.0.0

Example
EDF::Version versionDefault;

Version( const char* )

Example
EDF::Version version( "1.2.3" );

Version( const uint32_t& )

Example
EDF::Version versionValue( 10203 );

Version( const char (&str)[N] )

Example
const char array[] = "10.20.30";
EDF::Version versionArray( array );

Version( const EDF::String<N>& str )

Example
EDF::String<32> string( "123.456.789" );
EDF::Version versionString( string );

Access

getMajor()

Example
EDF::Version version( "1.2.3" );
version.getMajor(); // 1

getMinor()

Example
EDF::Version version( "1.2.3" );
version.getMinor(); // 2

getPatch()

Example
EDF::Version version( "1.2.3" );
version.getPatch(); // 3

asNumber()

Example
EDF::Version version( "1.2.3" );
version.asNumber(); // 10203

asString()

Example
EDF::Version version( "1.2.3" );
version.asString(); // EDF::String<9>("1.2.3")

asCString()

Example
EDF::Version version( "1.2.3" );
version.asCString(); // "1.2.3"

EDF Version

This is a static function to get the current version of EDF being used.

Example
EDF::Version::EDF();

Comparison Operators

operator>()

Example
EDF::Version version( "1.2.3" );
if( version > "0.1.0" ) {
    /* ... */
}
if( "0.1.0" > version ) {
    /* ... */
}
if( version > EDF::Version() ) {
    /* ... */
}

operator<()

Example
EDF::Version version( "1.2.3" );
if( version < "2.0.0" ) {
    /* ... */
}
if( "2.0.0" < version ) {
    /* ... */
}

operator>=()

Example
EDF::Version version( "1.2.3" );
if( version >= "1.1.0" ) {
    /* ... */
}
if( "0.1.0" >= version ) {
    /* ... */
}

operator⇐()

Example
EDF::Version version( "1.2.3" );
if( version <= "2.0.0" ) {
    /* ... */
}
if( "2.0.0" <= version ) {
    /* ... */
}

operator==()

Example
EDF::Version version( "1.2.3" );
if( version == "1.2.3" ) {
    /* ... */
}
if( "1.2.3" == version ) {
    /* ... */
}

operator!=()

Example
EDF::Version version( "1.2.3" );
if( version != "2.14.56" ) {
    /* ... */
}
if( "2.14.56" != version ) {
    /* ... */
}
